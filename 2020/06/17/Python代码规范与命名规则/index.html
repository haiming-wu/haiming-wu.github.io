Python代码规范与命名规则

主要涉及5类命名规则：模块、类名、函数、变量和常量，为了加深记忆，进一步和Java进行了对比。

1.模块

  • 模块尽量使用小写命名，首字母保持小写，尽量不要用下划线(除非多个单词，且数量不多的情况)

2.类名

  • 类名使用驼峰(CamelCase)命名风格，首字母大写，私有类可用一个下划线开头

3.函数

  • 函数名一律小写，如有多个单词，用下划线隔开
  • 私有函数在函数前加一个下划线_

4.变量

  • 变量名尽量小写, 如有多个单词，用下划线隔开
  • 常量采用全大写，如有多个单词，使用下划线隔开

5.常量

  • 常量使用以下划线分隔的大写命名

6.Python与Java对比

Python Java
模块/包 尽量使用小写命名，首字母保持小写，尽量不要用下划线(除非多个单词，且数量不多的情况) 每个字母都需要小写
类 类名使用驼峰(CamelCase)命名风格，首字母都大写，私有类可用一个下划线开头 首字母都大写，而且中间不使用任何的连接符。尽量使用英文。如ConnectionFactory
函数/方法 1）一律小写，如有多个单词，用下划线隔开
2）私有函数在函数前加一个下划线_		 首单词全部小写，如果一个方法由多个单词构成，那么从第二个单词开始首字母大写，不使用连接符。
变量/属性 1）变量名尽量小写, 如有多个单词，用下划线隔开
2）常量采用全大写，如有多个单词，使用下划线隔开		 与方法相同。如ageOfPerson
常量 常量使用以下划线分隔的大写命名 所有单词的字母都是大写，如果有多个单词，那么使用下划线链接即可。
